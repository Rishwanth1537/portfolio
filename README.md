## Project Overview

A fully responsive personal portfolio website built using pure HTML5 and CSS3 — no JavaScript frameworks, no libraries. It showcases my skills, projects, education, and contact information as an AI & Data Science student actively seeking an AI/ML internship.

The site features a dark red-and-black theme with smooth animations, CSS Grid and Flexbox layouts, hover effects, transitions, a CSS-only hamburger menu for mobile, and a custom SVG favicon.

### Sections
- **Home** — Hero section with name, tagline, and CTA
- **About** — Profile photo, personal bio
- **Skills** — Six skill category cards (Programming, ML, CV, NLP, Tools, Data)
- **Projects** — Four project cards with tags and GitHub links
- **Education** — Timeline cards for degrees and certifications
- **Contact** — Social links and a contact form

## Setup Instructions

No build tools, no dependencies, no installation required.

1. Clone or download the repository:
   ```bash
   git clone https://github.com/Rishwanth1537/portfolio.git
   ```
2. Make sure all three files (`index.html`, `new.css`, `Rishi.png`) are in the **same folder**.
3. Open `index.html` in any modern browser.

## Features Implemented

| Feature | Details |
|---|---|
| HTML5 Semantic Structure | `<nav>`, `<section>`, `<footer>` |
| CSS Variables | Consistent color theme via `:root` |
| CSS Grid | Skills, Projects, Education, Contact layouts |
| Flexbox | Nav, footer, contact links, pills |
| Animations | `fadeUp`, `fadeIn`, `slideDown` keyframes |
| Hover Effects | Cards lift, nav underline slides, pills shift |
| Smooth Scrolling | `scroll-behavior: smooth` on `html` |
| Responsive Design | Media queries at 900px, 640px, 480px |
| Hamburger Menu | CSS-only checkbox toggle for mobile |
| Gradient Background | Radial gradient on hero section |
| Card Layouts | Skill cards, project cards, edu cards |
| Box Shadows | Cards and image frame |
| Custom Favicon | Inline SVG, no external file needed |
| Profile Image | Responsive with zoom-on-hover effect |
